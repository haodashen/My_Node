 awk是操作数据并创建报告的语言，awk可以在命令行使用简单的运算符，也可以用于高级语言，awk可以操作数据，是Shell不可或缺的工具。


awk编程方式：1.文件输入
			awk ' pattern' filename
			awk '{action}' filename
			awk 'pattern {action}'filename
	    2.从指令输入
			command | awk 'pattern'
			command } awk '{action}'
			command | awk 'pattern {action}'



awk函数：（1）print函数：awk命令的动作（action）部分使用花括号（{}），若没有指定动作而匹配模式，awk就执行默认的动作，在屏幕上显示匹配行。使用print函数显示内容无格式化，printf函数可以使用参数变量计算出的值和字符串常数，字符创需要使用双引号（""），逗号(,)用于分隔参数。

  例：date | awk '{print "Current time:"$4}'

print函数的转移符号：\b:退格  \f:换页符 \n:换行符 \r:回车 \t:Tab  \c:c代表字符

（2）printf函数:printf函数与C语言中的printf语句一样，通过标准输出返回已格式化的字符串，printf语句拥有格式指示符和修饰符等控制字符串，控制字符串有逗号和逗号分隔的表达式的目录决定

printf格式指示符：%s:显示字符串 %x:显示16进制数 %o:显示八进制数
printf格式字符：c:字符 s:字符串 d:十进制赎数字
 例：echo "LINUX"|awk '{printf "|%-15s|\n",$1}'



awk内置变量：(1)OFMT变量控制数字输出格式，默认设置为%.6g
	     (2)ORS变量
	     (3)RS变
	     (4)MR变量
NF变量：当前记录的全部字段数
NR:记录数
FS:输入字段分隔符，默认值为空格
RS:输入记录分隔符
RT:记录结束符，设置为与字符或RS指定正则表达式相匹配的输入文本
FNR:当前文件的全部记录数
ORS:输出记录分隔符
OFS:输出字段分隔符
OFMT:数字输出格式
ARGC:命令行参数个数
ARGV:命令行参数数组
CONVFMT:数字格式转换

记录：awk不能查看输入数据，但可以查看格式和结构，默认每行称为"记录"
      (1)记录分隔符：默认设置的输入和输出分隔符是回车，保存在awk变量ORS和RS
      (2)$0变量：awk中的所有记录都浏览为$0,$0保存所有当前记录
      (3)NR变量：每个记录保存于awk内置变量中，记录保存后,NR的值加1

字段：每个记录默认由词构成，所谓的词由空格或Tab分隔的字段组成，每个词称为1个字段。awk保持内置变量NF的字段数，每行NF值一般        可拥有100个字段。

字段分隔符：awk内置变量FS拥有输入字段分隔符的值，FS默认值为空格或Tab，可以在BEGIN语句或命令行更改FS值，在命令行使用-F选项            更改FS值（awk -F:）
          指定多个字段分隔符：awk -F '[ :\t]',在（[]）可以使用正则表达式，本例子中使用空格，冒号（:）和Tab作为字段分隔符


模式：awk模式管理awk输入行的动作，该模式有正则表达式，真和假状态结果或二者结合组成
动作：awk动作是使用花括号（{}）的语句，已分好（;）分隔。模式在动作之前，动作可创建为简单语句或高级语句组，语句已分号（;）        或者newline分隔。格式{action}

BEGIN模式：awk处理输入文件的行之前执行BEGIN模式，并将其置于动作块之前。若awk未完成BEGIN动作块，则BEGIN块不读取输入内容，所	   以输入文件不存在也可以测试。BRGIN模式常常用于更改内置变量（OF，RS，FS等），分配用户自定义变量的初始值并作为输出的一部分用于打印有文件或标题
	例:awk 'BEGIN{FS=":";OFS="\t";ORS="\n\n"}{print $1,$2,$3}'filename
END模式:END模式不匹配任何输入行，但执行END模式相关动作，读取最后一个记录后显示行号



模式和动作可以连接起来，动作使用花括号，模式管理的动作范围从第一个前花括号到第一个后花括号。
无动作模式显示匹配模式的所有行，字段串匹配模式包含使用斜杠（/）的正则表达式。
格式：pattern{action statement;action statement;etc.}

变量：awk程序可以添加变量，若不初始化变量，则成为0或空格字符
数值：整数型和实数型
命令行自定义变量：awk -v key=value:使用-v选项：将指定的value赋值给key。例：awk -v month=9 year=2018 filename
